# 简介

AndroidLib 总结了项目中常用的库，实现Code更便捷的重用，优化项目结构，加速项目开发。

- 丰富的功能库
- 大量的测试用例
- 完整说明文档
- 实时更新


主要包含两部分：

1. appbase，主要是app常用的一些基类。使用的时候视情况copy过去即可（BaseActiviy, BaseView等）
2. utilslib，主要是常用的一些方法库
